# RawPixel 修图工具

基于 LibRaw-Wasm 开发的在线 RAW 格式图片处理工具，专注于为摄影爱好者提供便捷的相机原始文件编辑体验。

## 功能特点

- ✅ 原生支持 RAW 格式图片解析与处理
- ✅ 内置 AI 智能参数调整功能，一键优化图片效果
- ✅ 目前已支持索尼相机 RAW 格式文件
- ✅ 纯网页端运行，无需安装任何软件
- ✅ 实时预览修图效果，所见即所得

## 支持设备

当前版本已完成对以下品牌相机的支持：

- 索尼（Sony）系列相机 RAW 格式

更多相机品牌支持正在开发中，敬请期待！

## 使用方法

1. 访问工具网页
2. 上传你的 RAW 格式图片
3. 选择预设效果或使用 AI 智能调整
4. 微调参数至满意效果
5. 导出处理后的图片
